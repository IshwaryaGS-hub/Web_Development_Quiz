## Web development quiz app

A fast, interactive Quiz Application built using HTML, CSS, and JavaScript DOM Manipulation.
It includes a countdown timer of 240 seconds (4 minutes), live score tracking, next/previous navigation, and dynamic UI updates — all without any frameworks.

## Live Demo

🔗 Deployed on Vercel:
👉 https://web-development-quiz-mocha.vercel.app/

---- 

## Features

✔ Fully dynamic questions using pure JavaScript DOM Manipulation

✔ 240-second countdown timer (auto-submit when time ends)

✔ Live score calculation

✔ Clean and responsive UI

✔ Option selection highlight with instant feedback

✔ Works on all screen sizes (mobile, tablet, desktop)

✔ Zero dependencies — no external libraries needed


----- 

## Tech Stack

HTML5

CSS3

JavaScript (DOM Manipulation)

Vercel for hosting


##  How It Works
1. DOM-based rendering

Questions, options, and score are created dynamically using JavaScript.

2. 180-second countdown

A built-in timer:

let timeLeft = 180 ;


If timer reaches 0 → quiz auto-submits.

3. Real-time updates

When a question is answered, DOM elements update instantly without reload.
